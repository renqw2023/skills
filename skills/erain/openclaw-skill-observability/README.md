# OpenClaw Observability Skill

This skill provides basic observability tools for your OpenClaw instance.

## Features

- **Cost Reporting**: Check estimated API usage costs (Placeholder MVP).
- **Error Logs**: Quickly retrieve recent errors from system logs.

## Installation

This skill is typically installed in `~/clawd/skills/openclaw-skill-observability`.

## Usage

Ask OpenClaw:
- "Show me recent errors"
- "How much have I spent on API calls?"
