# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-10

### Added
- Initial release of OPNsense Admin Skill
- API helper script for common operations
- Automated backup script with retention
- Service control script via SSH
- Complete documentation for OPNsense 26.1+
- Support for Suricata IDS/IPS management
- Unbound DNS configuration support
- Firewall rules and NAT management

### Security
- API credentials via environment variables or secure file
- No hardcoded credentials
- Support for HTTPS with certificate validation
