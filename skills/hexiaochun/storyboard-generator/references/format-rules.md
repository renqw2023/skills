# 分镜生成规则详解

## 1. 镜头表字段规范

### 1.1 必填字段

| 字段 | 说明 | 格式 |
|------|------|------|
| ShotID | 镜头唯一标识 | E{集}_S{序号} |
| SceneID | 场次号 | 1-1, 1-2, 2-1 |
| 景别 | 镜头远近 | 远景/全景/中景/近景/特写 |
| 机位 | 摄像机角度 | 平视/俯视/仰视/越肩/主观 |
| 运镜 | 主导运动 | 静止/微推/拉远/轻摇/跟拍 |
| 目的 | 画面功能 | 信息/情绪/动作/反转/钩子 |
| 唯一信息点 | 本镜头传达的信息 | 一句话 |
| 定格瞬间 | 关键帧描述 | 一句话，可截图 |
| CharID | 主体角色 | @角色ID |
| SceneKey | 场景Key | scene_xxx |

### 1.2 选填字段

| 字段 | 说明 | 格式 |
|------|------|------|
| 动作链 | 2-4步可视化动作 | 动作1→动作2→动作3 |
| 台词/OS | 本镜头台词 | 台词内容或OS：内心独白 |
| 音效点 | 声音效果 | 环境音/拟音 |
| PropKey | 涉及道具 | prop_xxx |
| 优先级 | 重要程度 | P0/P1/P2 |
| 状态 | 制作进度 | todo/ok/fix |
| 连续性备注 | 与前后镜头衔接 | 文字描述 |

## 2. 景别详解

### 2.1 景别定义

| 景别 | 画面范围 | 功能 |
|------|----------|------|
| 远景 | 广阔环境，人物极小 | 建立环境规模 |
| 全景 | 人物全身+周围环境 | 展示人物与环境关系 |
| 中景 | 人物膝盖以上 | 展示动作、互动 |
| 近景 | 人物胸部以上 | 展示表情、情绪 |
| 特写 | 面部或物品局部 | 强调细节、关键信息 |

### 2.2 景别选择指南

| 场景类型 | 推荐景别 | 原因 |
|----------|----------|------|
| 场景建立 | 远景/全景 | 交代环境 |
| 日常对话 | 中景/近景 | 展示互动 |
| 情绪爆发 | 近景/特写 | 强调表情 |
| 动作场面 | 全景/中景 | 展示动作 |
| 关键道具 | 特写 | 强调重要性 |

### 2.3 景别转换规则

- 避免景别跳跃过大（远景→特写）
- 同一场景内景别应有变化
- 重要信息点用近景/特写强调

## 3. 机位详解

### 3.1 机位定义

| 机位 | 摄像机位置 | 情绪效果 |
|------|------------|----------|
| 平视 | 与人物视线平齐 | 中性、客观 |
| 俯视 | 从上往下拍 | 弱势、渺小、俯瞰 |
| 仰视 | 从下往上拍 | 强势、威严、崇敬 |
| 越肩 | 从一人肩后拍另一人 | 代入、对峙 |
| 主观 | 角色视角 | 沉浸、紧张 |

### 3.2 机位选择指南

| 场景类型 | 推荐机位 | 原因 |
|----------|----------|------|
| 日常场景 | 平视 | 自然客观 |
| 强者出场 | 仰视 | 突显威压 |
| 角色困境 | 俯视 | 表现弱势 |
| 对话场景 | 越肩 | 代入感强 |
| 发现敌人 | 主观 | 紧张沉浸 |

## 4. 运镜详解

### 4.1 运镜定义

| 运镜 | 摄像机运动 | 效果 |
|------|------------|------|
| 静止 | 不动 | 稳定、客观 |
| 微推 | 缓慢推近 | 强调、紧张 |
| 拉远 | 缓慢拉远 | 揭示、结束 |
| 轻摇 | 水平/垂直移动 | 扫视、跟踪 |
| 跟拍 | 跟随主体移动 | 动感、紧张 |

### 4.2 运镜选择规则

- 每镜头只选 1 个主导运镜
- 静止为主，运动为辅
- 重要时刻用微推强调
- 动作场面可用跟拍

## 5. 画面目的标签

### 5.1 标签定义

| 标签 | 功能 | 示例 |
|------|------|------|
| 信息 | 传达关键信息 | 发现敌人、看到道具 |
| 情绪 | 展示角色情绪 | 惊讶、愤怒、悲伤 |
| 动作 | 展示动作过程 | 拔剑、攻击、躲避 |
| 反转 | 剧情反转点 | 真相揭露、身份暴露 |
| 钩子 | 制造悬念 | 神秘人影、留下伏笔 |

### 5.2 标签使用规则

- 每镜头只选 1 个标签
- 一场戏应包含多种标签
- 重要转折点用"反转"
- 场景结尾用"钩子"

## 6. 唯一信息点规则

### 6.1 规则
- 每镜头只传达 1 个信息
- 用一句话概括
- 必须具体明确

### 6.2 正确示例
```
✓ 陈风发现身后有敌人
✓ 温晴露出惊讶的表情
✓ 青锋剑发出耀眼光芒
```

### 6.3 错误示例
```
✗ 陈风发现敌人并开始战斗（两个信息）
✗ 场面很紧张（不具体）
✗ 剧情发展（太模糊）
```

## 7. 定格瞬间规则

### 7.1 规则
- 描述一个可截图的画面
- 必须具体可视化
- 包含人物动作/表情

### 7.2 正确示例
```
✓ 陈风猛然转身，眼神警觉，手已按在剑柄上
✓ 温晴睁大双眼，嘴唇微张，露出惊讶神色
✓ 青锋剑出鞘瞬间，剑身青光一闪
```

### 7.3 错误示例
```
✗ 陈风很警觉（不具体）
✗ 温晴的反应（没有描述）
✗ 剑很厉害（不可视化）
```

## 8. 提示词三段式规范

### 8.1 结构
```
- 风格：{STYLE_BASE一句话}
- 主体：{@角色ID形象} + {本镜头表情动作}
- 画面：{SceneKey场景} + {景别机位} + {构图} + {定格瞬间}
```

### 8.2 风格段规则
- 必须引用 STYLE_BASE 或 STYLE_VAR
- 逐字复制，不得修改
- 回忆场景用对应 STYLE_VAR

### 8.3 主体段规则
- 只允许 1 个主主体
- 配角只能虚化/剪影/轮廓
- 引用角色卡形象描述
- 添加本镜头表情动作增量

### 8.4 画面段规则
- 引用场景卡描述
- 明确景别和机位
- 描述构图（主体位置、遮挡、前景后景）
- 写清定格瞬间

### 8.5 示例
```
- 风格：2.5D国风动画，青绿山水色调，细腻工笔材质，干净利落线条，柔光侧逆光，仙侠清冷氛围
- 主体：陈风 (@jl_chenfeng) 20岁男性，剑眉星目，黑发束冠，白色仙袍金边点缀，腰佩青锋剑，猛然转身，眼神警觉，右手按在剑柄上
- 画面：妖兽谷，嶙峋怪石，黑雾缭绕，近景，平视，主体居中，背景虚化，定格在转身警觉的瞬间
```

## 9. 导演三帧法规范

### 9.1 三帧定义

| 帧 | ID后缀 | 功能 | 目的标签 |
|----|--------|------|----------|
| 起帧 | KF_A | 建立环境与关系 | 信息 |
| 爆帧 | KF_B | 冲突/动作爆点 | 动作/反转 |
| 钩帧 | KF_C | 悬念/转场 | 钩子/情绪 |

### 9.2 三帧必须包含
- 镜头ID
- 目的标签
- 唯一信息点
- 定格瞬间
- 景别/机位
- 三段式提示词

## 10. 18镜头覆盖规范

### 10.1 分布建议
```
第1场（6镜头）
├── 建立镜头 ×1
├── 信息镜头 ×1
├── 动作镜头 ×2
├── 反应镜头 ×1
└── 转场镜头 ×1

第2场（6镜头）
├── 同上结构

第3场（6镜头）
├── 同上结构
```

### 10.2 结构位检查
- [ ] 有建立镜头（环境/关系）？
- [ ] 有信息镜头（关键道具/信息）？
- [ ] 有动作镜头（主要动作）？
- [ ] 有反应镜头（情绪反应）？
- [ ] 有转场镜头（切出/悬念）？

## 11. 负向提示词模板

### 11.1 通用负向
```
文字、水印、logo、边框、二维码、低清晰度、模糊、噪点、过曝、透视错误、肢体畸形、多余手指、画面脏乱
```

### 11.2 古风负向
```
现代建筑、现代服饰、枪械、车辆、电线杆、玻璃幕墙
```

### 11.3 现代负向
```
古代建筑、古代服饰、盔甲、长袍、魔法光效
```

### 11.4 安全内容负向
```
血腥特写、内脏、喷溅、恐怖猎奇、露骨内容
```

## 12. QC 检查规范

### 12.1 P0（必须修复）
- 缺建立/反应/转场镜头
- 定格瞬间不可截图
- 轴线跳跃
- 信息点不清晰
- 主体超过1个

### 12.2 P1（第二优先）
- 角色脸/服装漂移
- 道具与角色卡不一致
- 场景锚点漂移

### 12.3 P2（最后修复）
- 画面不够干净
- 光影不统一
- 不够电影化

## 13. 返修规范

### 13.1 返修顺序
```
P0 → P1 → P2
```

### 13.2 返修原则
- 先改镜头表字段，再改提示词
- 一次只做一个主手术刀

### 13.3 主手术刀选择
| 问题 | 手术刀 |
|------|--------|
| 信息点不清 | 改构图 |
| 爆点不落 | 改定格瞬间 |
| 可剪性差 | 改景别机位 |
| 一致性差 | 改道具/表情 |
