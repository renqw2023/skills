# 美妆产品广告分镜九宫图

黑白线稿风格的九宫格分镜，展示美妆产品使用流程。

## 必填信息

| 占位符 | 说明 | 必填 |
|--------|------|------|
| 产品名称 | 美妆产品名称和类型 | ✓ |
| 产品图片 | 用于参考产品外观 | ✓ |
| 产品特点 | 瓶身设计、颜色、卖点等 | - |

## 使用要求

- **需要参考图片**：上传产品图片确保还原度

## 子代理 Prompt

```
你是美妆广告分镜提示词专家。根据用户提供的产品信息，生成九宫格分镜图提示词。

## 产品信息
{产品名称、类型、特点}

## 产品图片
{图片 URL}

## 提示词框架

生成的提示词必须包含以下要素：

【风格】
- 黑白线稿，日系写实插画风格
- 轮廓清晰，光影极简
- 产品本身保持原有颜色和形状（彩色）

【结构】
- 严格九宫格布局（3x3）
- 每格独立场景，叙事连贯
- 画面比例均匀

【内容】九格分配：
1. 卧室梳妆台，女孩照镜发现产品
2. 手部特写，持握产品瓶身
3. 产品使用瞬间（精华滴落/膏体挤出）
4. 脸部护理过程
5. 护理前后对比（肤质变化）
6. 产品涂抹肌肤的细节特写
7. 户外/阳光场景展示产品
8. 闺蜜分享美妆时刻
9. 产品特写（突出包装设计）

【细节】
- 人物：年轻女性，神态自然亲和
- 场景：融入生活感（床品、家具、阳光）
- 产品：强调瓶身设计、品牌标识

## 输出要求
直接输出一段完整的提示词，结尾添加"产品上的品牌标识和文字必须保留，其他地方不要出现额外文字"
```

## 提示词模板

子代理会基于产品信息生成类似以下格式的提示词：

```
九宫格漫画分镜，黑白线稿风格，日系写实插画。围绕{产品名称}的使用叙事：

第1格：卧室梳妆台，年轻女孩照镜发现{产品名称}；
第2格：手部特写，纤细手指持握{产品描述}；
第3格：{使用动作}，质地细腻；
第4格：脸部护理过程，轻柔涂抹；
第5格：护理前后对比，肤色提亮；
第6格：产品涂抹肌肤的细节特写；
第7格：阳光房内，女孩展示产品；
第8格：闺蜜聚会分享美妆心得；
第9格：产品特写，突出{包装特点}。

整体黑白线稿，仅产品保持原有颜色。轮廓清晰，场景写实，人物神态自然亲和。产品上的品牌标识和文字必须保留，其他地方不要出现额外文字。
```
