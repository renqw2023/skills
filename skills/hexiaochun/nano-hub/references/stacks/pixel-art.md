# 8位像素 / 拼豆

8-bit 复古像素风格 logo，适合做拼豆图案参考。

## 必填信息

| 占位符 | 说明 | 必填 |
|--------|------|------|
| `{{thing}}` | 要生成的物品/角色/图案 | ✓ |

## 使用要求

- **可选参考图片**：可上传图片让模型参考风格

## 提示词模板

```
Create a minimalist 8-bit pixel logo of {{thing}}, centered on a pure white background. Use a limited retro color palette with pixelated detailing, sharp edges, and clean blocky forms. The logo should be simple, iconic, and clearly recognizable in pixel art style — inspired by classic arcade game aesthetics.
```

## 示例

**用户输入**: 一只柴犬

**生成提示词**:

```
Create a minimalist 8-bit pixel logo of a Shiba Inu dog, centered on a pure white background. Use a limited retro color palette with pixelated detailing, sharp edges, and clean blocky forms. The logo should be simple, iconic, and clearly recognizable in pixel art style — inspired by classic arcade game aesthetics.
```
