# 情绪语气词映射

使用 `speech-2.8-hd` 模型时，可通过语气词标签增强语音的情感表达。

## 语气词对照表

| 情绪 | 语气词 | 使用示例 |
|-----|-------|---------|
| 开心/高兴 | `(laughs)` | 太棒了(laughs)！ |
| 轻笑 | `(chuckle)` | 是啊(chuckle)，我也这么想。 |
| 惊讶 | `(gasps)` | 真的吗(gasps)？ |
| 感慨/无奈 | `(sighs)` | 唉(sighs)，算了吧。 |
| 悲伤/哭泣 | `(crying)` | 我好难过(crying)... |
| 抽泣 | `(sniffs)` | 我好难过(sniffs)。 |
| 思考 | `(emm)` | (emm)让我想想... |
| 喘息/感叹 | `(breath)` | 终于完成了(breath)！ |
| 紧张/气喘 | `(pant)` | 怎么办(pant)？ |
| 清嗓 | `(clear-throat)` | (clear-throat)各位好。 |

## 预处理示例

### 原文

```
小明：今天考试终于结束了！太开心了！
小红：是啊，我感觉考得还不错。你呢？
小明：嗯...让我想想，应该还行吧。
```

### 处理后

```
小明：今天考试终于结束了(breath)！太开心了(laughs)！
小红：是啊(chuckle)，我感觉考得还不错。你呢？
小明：(emm)让我想想(breath)，应该还行吧。
```

## 使用规则

1. **位置灵活**：语气词可放在句首、句中或句尾
2. **适度使用**：每段对话 1-2 个语气词为宜，过多会显得不自然
3. **情感匹配**：语气词应与对话情绪标签一致
4. **自然过渡**：相邻对话的情绪过渡要自然

## 情绪强度调节

根据对话情绪标签，调整语气词的使用：

- **强烈情绪** (非常开心/悲伤/愤怒)：可用 2-3 个语气词
- **一般情绪** (开心/难过/惊讶)：使用 1-2 个语气词
- **平和情绪** (陈述/询问)：可不用或仅用 `(breath)`

## 停顿控制

使用 `<#x#>` 标记控制对话间的停顿时长：

| 场景 | 停顿时长 | 标记示例 |
|-----|---------|---------|
| 快节奏对话 | 0.3 秒 | `<#0.3#>` |
| 正常对话 | 0.5 秒 | `<#0.5#>` |
| 沉思/感慨 | 1.0 秒 | `<#1#>` |
| 戏剧性停顿 | 1.5-2 秒 | `<#1.5#>` |
