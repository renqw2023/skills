# 风格提取规则详解

## 1. 风格一句话构成

```
{画风类型}，{色调倾向}，{材质质感}，{线稿风格}，{光影特点}，{氛围基调}
```

### 1.1 画风类型库

| 画风 | 特点 | 适用题材 |
|------|------|----------|
| 2.5D国风动画 | 立体感+国风元素 | 仙侠、武侠、历史 |
| 日系赛璐璐 | 平涂、色块分明 | 校园、恋爱、热血 |
| 韩漫写实风 | 精致、真实感强 | 都市、霸总、复仇 |
| 美漫厚涂风 | 厚重、光影强烈 | 超英、动作、科幻 |
| 水墨写意风 | 留白、意境优先 | 古风、文艺、禅意 |
| 赛博朋克风 | 霓虹、科技感 | 科幻、反乌托邦 |
| 暗黑哥特风 | 阴暗、压抑 | 恐怖、悬疑、黑暗 |
| Q版可爱风 | 萌化、简化 | 搞笑、治愈、儿童 |

## 2. STYLE_VAR 变体规则

### 2.1 变体类型

| 变体类型 | 触发场景 | delta 示例 |
|----------|----------|------------|
| flashback | 回忆/闪回 | 降饱和、暖黄、柔焦 |
| dream | 梦境/幻觉 | 高光晕、边缘模糊、饱和度极低 |
| system_ui | 系统界面 | 青蓝发光、科技线条、半透明 |
| battle | 战斗/高燃 | 高对比、动态模糊、粒子特效 |
| horror | 恐怖/压迫 | 去饱和、冷色、底光、噪点 |

### 2.2 delta 写法规范

**正确写法**（只写变化项）：
```
delta：
- 色彩：饱和度 -30%，色温 +500K
- 对比：更柔和
- 画面质感：边缘暗角 +20%
```

**错误写法**（重写 BASE）：
```
delta：
- 画风：2.5D国风动画  ❌ 不应重写
- 色调：暖黄复古     ❌ 应写增量
- 线稿：干净利落     ❌ 不应重写
```

## 3. 内容边界（禁忌）模板

### 3.1 古风题材禁忌
```
- 现代建筑、电线杆、玻璃幕墙
- 现代服饰、牛仔裤、运动鞋、T恤
- 现代交通工具、汽车、飞机
- 现代电子设备、手机、电脑
- 现代发型、染发、爆炸头
```

### 3.2 现代题材禁忌
```
- 古代服饰、盔甲、长袍
- 古代建筑、宫殿、城墙
- 魔法、仙术、飞行
- 妖怪、神兽、仙人
```

### 3.3 通用禁忌（所有题材）
```
- 文字、水印、logo、边框
- 血腥内脏、残肢断臂
- 肢体畸形、多余手指
- 透视错误、比例失调
- 低清晰度、模糊、噪点
```

## 4. 镜头语言默认值模板

### 4.1 仙侠/武侠
```
镜头语言默认值：
- 景别倾向：中近景为主，山水全景建立意境
- 焦段倾向：50-85mm，人像舒适
- 景深倾向：浅景深，云雾虚化
- 运动克制：静止为主，动作戏跟拍
```

### 4.2 都市/现代
```
镜头语言默认值：
- 景别倾向：中景为主，城市远景建立环境
- 焦段倾向：35-50mm，空间感强
- 景深倾向：中景深，环境清晰
- 运动克制：轻微手持感，更真实
```

### 4.3 恐怖/悬疑
```
镜头语言默认值：
- 景别倾向：近景特写为主，局部不安
- 焦段倾向：24-35mm，略变形
- 景深倾向：极浅景深，焦外不安
- 运动克制：缓慢推进，压迫感
```

## 5. 风格一致性检查

### 5.1 自检清单
- [ ] 风格一句话是否 ≤30 字？
- [ ] 是否包含所有 6 个要素？
- [ ] 是否可直接粘贴到提示词？
- [ ] STYLE_VAR 是否只写 delta？
- [ ] 内容边界是否明确完整？
- [ ] 是否与题材调性匹配？

### 5.2 常见错误
1. 风格描述过长，无法粘贴
2. 缺少关键要素（如光影、氛围）
3. STYLE_VAR 重写 BASE 内容
4. 内容边界与题材矛盾
