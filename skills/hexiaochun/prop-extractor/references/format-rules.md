# 道具提取规则详解

## 1. 道具卡结构规范

每个道具必须包含以下字段：

```
- **PropKey：**prop_{项目}_{道具拼音}
- **道具名：**{道具名称}
- **类型：**{武器/信物/法宝/容器/日用/线索}
- **归属角色：**@{角色ID}（或"无归属/公共道具"）

### 外观描述
### 剧情意义
### 使用场景
### 道具提示词
```

## 2. 道具分类

| 类型 | 定义 | 提取优先级 | 示例 |
|------|------|------------|------|
| 武器 | 攻击/防御用具 | 高 | 剑、刀、弓、盾、法杖 |
| 信物 | 情感/身份象征 | 高 | 玉佩、戒指、信笺、照片 |
| 法宝 | 特殊能力物品 | 高 | 仙丹、符咒、法器、神器 |
| 线索 | 推动剧情的物品 | 高 | 地图、密信、证据、钥匙 |
| 容器 | 装载物品的器具 | 中 | 酒壶、药瓶、宝箱、背包 |
| 日用 | 日常使用物品 | 低 | 书籍、乐器、工具、食物 |

## 3. 道具筛选标准

### 3.1 必须提取
- 角色标志性物品（如主角的武器）
- 多次出现的物品
- 有特写镜头的物品
- 推动剧情的关键道具
- 有情感意义的物品

### 3.2 可以忽略
- 只出现一次的普通物品
- 纯背景装饰物品
- 无剧情意义的日用品
- 群演角色的通用物品

## 4. 外观描述规范

### 4.1 描述顺序
```
材质 → 颜色 → 尺寸 → 装饰 → 特效
```

### 4.2 材质库

| 材质类型 | 描述词 |
|----------|--------|
| 金属 | 寒铁、精钢、青铜、黄铜、白银、黄金 |
| 玉石 | 羊脂白玉、翡翠、碧玉、墨玉、玛瑙 |
| 木质 | 紫檀、黄花梨、乌木、桃木、柳木 |
| 布料 | 锦缎、丝绸、粗布、麻布 |
| 皮革 | 牛皮、羊皮、蛇皮、鳄鱼皮 |
| 其他 | 骨质、陶瓷、纸张、水晶 |

### 4.3 颜色库

| 颜色 | 描述词 |
|------|--------|
| 基础色 | 黑、白、红、蓝、绿、黄、紫 |
| 金属色 | 金色、银色、青铜色、古铜色 |
| 特殊色 | 透明、半透明、渐变、彩虹 |
| 发光色 | 泛青光、泛红光、金光闪烁 |

### 4.4 尺寸描述

| 尺寸参照 | 示例 |
|----------|------|
| 具体数值 | 三尺三寸（长剑）、拳头大小 |
| 身体比例 | 拇指大小、巴掌大、手臂长 |
| 相对大小 | 比常规剑长三寸、半人高 |

### 4.5 装饰描述

| 装饰类型 | 描述词 |
|----------|--------|
| 雕刻 | 龙纹、凤纹、云纹、花纹 |
| 镶嵌 | 镶嵌宝石、嵌入玉片 |
| 缠绕 | 缠绕丝带、绕有金线 |
| 悬挂 | 挂有流苏、系有玉坠 |

### 4.6 特效描述

| 特效类型 | 描述词 |
|----------|--------|
| 发光 | 光芒四射、微微发光、流光闪烁 |
| 流动 | 液体流动、光纹流转、符文浮现 |
| 飘动 | 剑穗飘动、雾气缭绕、火焰燃烧 |
| 变化 | 颜色变化、形态变化、体积膨胀 |

## 5. 剧情意义分类

| 意义类型 | 说明 | 示例 |
|----------|------|------|
| 身份象征 | 代表角色身份地位 | 令牌、官印、族徽 |
| 情感寄托 | 承载情感记忆 | 定情信物、遗物、礼物 |
| 能力来源 | 提供特殊能力 | 神器、法宝、功法秘籍 |
| 关键线索 | 推动剧情发展 | 地图、密信、证据 |
| 交易货币 | 用于交换价值 | 灵石、金币、丹药 |
| 保命手段 | 危急时刻使用 | 护身符、逃跑道具 |

## 6. PropKey 命名规范

### 6.1 格式
```
prop_{项目缩写}_{道具拼音或英文}
```

### 6.2 命名规则
- 全小写
- 下划线分隔
- 简洁明了
- 唯一不重复

### 6.3 示例
```
prop_jl_qingfengjian     # 剑来-青锋剑
prop_jl_yupei            # 剑来-玉佩
prop_jl_cangbaotu        # 剑来-藏宝图
prop_dushi_hetong        # 都市-合同
prop_dushi_shoubiao      # 都市-手表
```

## 7. 道具提示词生成规则

### 7.1 基础模板
```
{STYLE_BASE}，{道具名}，{材质}，{颜色}，{尺寸}，{装饰}，{特效}，白色背景，物品特写，居中构图，高清细节，无人物
```

### 7.2 示例
```
2.5D国风动画，青绿山水色调，细腻工笔材质，青锋剑，寒铁锻造，青蓝色剑身，银白剑锋，三尺三寸长剑，剑柄缠白色丝带，剑穗飘逸，剑身青光流转，白色背景，物品特写，居中构图，高清细节，无人物
```

## 8. 道具归属规则

### 8.1 单一归属
```
归属角色：@jl_chenfeng
```
道具始终属于一个角色，即使暂时被他人持有。

### 8.2 共同归属
```
归属角色：@jl_chenfeng / @jl_ningyu
```
道具为一对，分别属于两人（如鸳鸯佩）。

### 8.3 无归属
```
归属角色：无归属/公共道具
```
消耗品、公共物品、环境道具。

### 8.4 归属变化
```
归属角色：@jl_chenfeng（原@jl_shifu）
```
道具归属在剧情中发生转移。

```

- **PropKey：**prop_{项目}_{名称}
- **道具名：**{武器名}
- **类型：**武器
- **归属角色：**@{角色ID}

### 外观描述
- 材质：{锻造材料}
- 颜色：{主体颜色}
- 尺寸：{具体尺寸}
- 装饰：{剑柄/剑穗/剑鞘}
- 特效：{发光/特殊能力视觉}

### 武器属性（如有系统）
- 等级：{武器等级}
- 攻击力：{数值}
- 特殊效果：{效果描述}

### 剧情意义
- {意义1}
- {意义2}

- {场次1}
- {场次2}

{完整提示词}
```

## 10. 质量检查清单

- [ ] 是否所有道具都有唯一 PropKey？
- [ ] 外观描述是否按顺序（材质/颜色/尺寸/装饰/特效）？
- [ ] 外观描述是否足够具体可视化？
- [ ] 归属角色是否使用正确的角色 ID？
- [ ] 剧情意义是否明确？
- [ ] 使用场景是否列出主要场次？
- [ ] 道具提示词是否可直接用于生图？
- [ ] 是否漏提取关键道具？
