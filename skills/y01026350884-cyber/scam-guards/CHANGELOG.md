# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-10
### Added
- Full security pipeline: `scan_skill`, `verify_url`, `check_wallet`, `monitor_agent`, `evidence_chain`, and `report_generator`.
- Offline-first malicious pattern database (341+ indicators).
- PHI Lite behavioral analysis engine (Authority, Urgency, Isolation).
- SHA-256 legal evidence chaining for audit trails.
- Comprehensive reference documentation for threats and response playbooks.
- Multi-language comments (Korean/English) for core security logic.

## [0.1.0] - 2026-02-10
- Initial repository structure and core documentation.
- Project mission and Shield Layer architecture definition.
- Support for ClawHub skill specification (SKILL.md).
