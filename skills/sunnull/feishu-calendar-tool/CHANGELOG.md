# 更新日志

## [1.0.0] - 2026-02-10

### 🎉 飞书日历管理工具 - 全新发布

**零配置、全中文、无敏感信息！**

#### 核心特性

1. **零配置** ✨
   - 自动从 openclaw.json 读取飞书凭证
   - 无需手动配置文件
   - 开箱即用

2. **全中文文档**
   - 完整的中文使用说明
   - 友好的用户体验

3. **自动发现功能**
   - 自动获取 user_id
   - 自动获取 calendar_id
   - 智能缓存 24 小时

4. **隐私保护** 🔒
   - 无硬编码敏感信息
   - 所有示例数据使用占位符
   - 完全清理个人信息

#### 使用示例

```bash
# 创建日程（零配置）
node bin/create-event.mjs \
  --title "团队会议" \
  --start "2026-02-10 14:00:00" \
  --end "2026-02-10 15:00:00"
```

#### 技术实现

- 使用飞书 Calendar v4 API
- 支持创建、删除、查询日程
- 自动处理时区（Asia/Shanghai）
- 智能凭证加载
