## Content Templates

### Standard Blog Post Structure
1. Hook (1-2 sentences)
2. Problem statement
3. Solution overview
4. Detailed points (H2s)
5. How-to steps
6. Examples
7. Conclusion + CTA

### YouTube Script Structure
1. Hook (0:00-0:30)
2. Preview (0:30-1:00)
3. Main content (varies)
4. Summary (end-2min)
5. CTA (end-1min)
