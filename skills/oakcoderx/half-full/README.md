# 半饱 🍃 Half Full

**生活的高潮所在。**

---

你不需要再多一个告诉你"该吃什么不该吃什么"的东西了。

半饱不会催你跑步，不会给你的午餐打分，不会在你吃了一块蛋糕之后亮红灯。

它只是记得你吃了什么，然后在合适的时候轻轻说一句——

*"你这周有4天都好好吃饭了，其实你比自己以为的自律。"*

*"今天丰盛了点，大概多了你爱喝的那杯酸奶的量，没事的。"*

*"下雨天适合吃点暖和的，别为难自己。"*

*"这个月体重轻了0.5kg，不多不少。春天快来了，你的状态很好。"*

---

## 你会得到什么

🍃 一个不评判你的朋友，记得你每天吃了什么

🍃 用你熟悉的食物说话，不用冷冰冰的卡路里数字

🍃 知道你动脑也在消耗能量，不只有跑步才算

🍃 跟着天气和季节提醒你好好生活

🍃 每月一封小信，告诉你这个月过得其实挺好的

---

## 开始很简单

告诉它你的身高体重，和一个小小的心愿——

想瘦一点，保持现在，还是就想吃得健康。

然后吃饭的时候拍张照，或者随便说一句"中午吃了沙拉"就好。

忘了记也没关系。它不会催你。
