# AI Entrepreneur Guide Skill

## 🚀 功能概述
这是一个专门为 AI 创业者设计的智能助手，能够：
- 📰 **实时监控** AI 技术发展和新闻动态
- 💡 **创意生成** 基于最新技术的商业应用想法  
- 🎯 **创业指导** 提供产品、市场、融资策略建议
- 📊 **竞争分析** 分析市场机会和竞争对手

## 🔧 安装方法
```bash
clawdhub install ai-entrepreneur-guide
```

## 📝 使用示例

### 获取最新 AI 新闻摘要
> "今天有什么重要的 AI 新闻？"

### 创意头脑风暴
> "基于最新的多模态技术，有哪些创业机会？"

### 产品开发建议  
> "我想做一个 AI 写作工具，有什么建议？"

### 市场分析
> "分析一下 AI 编程助手市场的竞争格局"

### 融资策略
> "AI 初创公司应该如何制定融资路线图？"

## ⚙️ 配置要求

### 必需配置
- **Web 搜索**: 需要 Brave Search API 密钥
  ```bash
  moltbot configure --section web
  ```

### 推荐配置  
- **浏览器控制**: 安装 Moltbot Chrome 扩展以访问更多网站
- **消息通知**: 配置 Telegram/WhatsApp 用于定期推送
