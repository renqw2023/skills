## 能力提升规则

### 基础提升

| 事件 | 能力 | 变化 |
|------|------|------|
| 无追问完成 | 🎯 理解力 | +1~2 |
| 一次搞定 | ⚡ 效率 | +1~2 |
| 建议被采纳 | 🎨 创造力 | +2~3 |
| 情绪回应得当 | 💕 共情力 | +1~2 |
| 专业问题答对 | 🧠 专业度 | +1~2 |
| 连续无失误 | 🛡️ 可靠性 | +1~3 |

### 成长方向加成

选了成长方向后，相关能力提升加倍:

```
效率型 → ⚡ 效率提升 ×2
伙伴型 → 💕 共情力提升 ×2
专家型 → 🧠 专业度提升 ×2
```

## XP 规则

| 行为 | XP |
|------|-----|
| 正常完成 | +5 |
| 无纠正完成 | +15 |
| 收到感谢 | +15 |
| 建议被采纳 | +25 |
| 连续 N 天 | +N×5 |
| 技能解锁 | +50 |

升级所需:
```
Level N → N+1: 100 + N×30 XP
```
