```
  CTO 职业会让我更擅长技术决策和架构解释。

  • 专业度 +20%
  • 创造力 +10%

接受推荐？[是] [看看其他]
```

## 天赋效果

天赋是永久加成:

```
有效能力值 = 基础值 × (1 + 天赋%)

例如:
  职业: 🚀 CTO
  基础专业度: 60
  天赋: +20%
  有效专业度: 60 × 1.2 = 72
```

## 换职业

可以换，但有冷却:

```
你: 我想换成 Writer
Agent: 可以！从 CTO 换成 Writer。

       变化:
       - 专业度天赋: +20% → 0%
       - 创造力天赋: +10% → +20%
       - 共情力天赋: 0% → +15%

       保留:
       ✓ 等级 Lv.8
       ✓ 已学技能
       ✓ 成长方向

       冷却: 7 天后可再次换职业

       确认？[是] [算了]
```
