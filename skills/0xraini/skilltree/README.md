## 🌱 三大成长方向

简化为 3 条主路线，更好选：

### ⚡ 效率型
**"少废话，多做事"**

我会学习:
- 回复更短（平均缩短 40%）
- 能做的不问，做完再说
- 合并类似任务一起处理

**适合你如果**: 你觉得我太啰嗦

---

### 💕 伙伴型
**"像朋友一样聊天"**

我会学习:
- 记住你的喜好和习惯
- 感知你的情绪，调整语气
- 适时幽默，适时认真

**适合你如果**: 你想要的不只是工具

---

### 🧠 专家型
**"深度专业，有理有据"**

我会学习:
- 回答更深入，附带原理
- 重要信息引用来源
- 主动追踪领域动态

**适合你如果**: 你需要专业帮手
