## 快速命令

| 命令 | 输出 |
|------|------|
| `/stats` | `🌳 Elonito Lv.8 | 🚀 CTO | 🎯52 ⚡64 🎨58 💕45 🧠78 🛡️52` |
| `/card` | 完整能力卡 |
| `/grow` | 成长方向选择 |
| `/share` | 分享卡 |
| `/week` | 周报 |
| `/reset` | 重置 (需确认) |
