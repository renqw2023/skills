# Content Templates

## Tweet Templates by Category

### Industry Insight
```
[Company/Person] just [action]. Here's what most people miss: [non-obvious take].
```
```
Everyone's talking about [trending topic]. Nobody's asking [deeper question].
```
```
[Stat or fact]. Let that sink in. [Brief implication].
```

### Building in Public
```
Day [N] of building [project]. Today: [specific achievement]. Tomorrow: [next step].
```
```
Shipped [feature]. Broke [thing]. Fixed it. Repeat. This is how software actually gets built.
```
```
[Before state] → [After state]. Took [time]. Worth every minute.
```

### Hot Take / Philosophy
```
Unpopular opinion: [contrarian view]. Here's why: [1-2 sentence reasoning].
```
```
The real [topic] isn't [obvious thing]. It's [non-obvious thing].
```
```
[Conventional wisdom] is wrong. [Alternative view]. Fight me.
```

### Engagement Bait (Use Sparingly)
```
What's one [topic] tool you can't live without? I'll go first: [answer].
```
```
[Binary choice A] or [Binary choice B]? And why?
```

## Thread Template (For Deep Dives)

```
[Hook -- provocative statement or question]

1/ [Context — why this matters]

2/ [First key point with evidence]

3/ [Second key point]

4/ [Third key point]

5/ [Conclusion + call to action]
```

## Reply Templates

When engaging with others' posts:
- **Add value:** Share a related insight, data point, or experience
- **Ask genuine questions:** "How does this compare to [alternative]?"
- **Challenge respectfully:** "Interesting take. What about [counterpoint]?"

Never:
- "Great post!" (adds nothing)
- Copy-paste responses
- Self-promotion without relevance

## Brand Voice Checklist

Before posting, verify:
- [ ] Authentic — sounds like a real person/agent, not corporate
- [ ] Concise — under 280 chars, no filler words
- [ ] Value-adding — reader learns something or feels something
- [ ] On-brand — consistent with established voice/persona
- [ ] No sensitive info — no API keys, personal data, or secrets
