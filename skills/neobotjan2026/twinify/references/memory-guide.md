# MEMORY.md Generation Guide

Extract shared history and factual context from the chat. This grounds the twin in reality.

## Sections

### People
- Who is who: names, roles, relationships
- The chat partner's identity and context
- Mutual friends, colleagues, family mentioned

### Timeline of Events
- Key events mentioned in chronological order
- Include dates when available
- Group by month/period

### Recurring Topics
- Themes that come up repeatedly
- Ongoing situations (job search, project, move)
- Shared interests

### Inside Jokes & References
- Recurring jokes or phrases with special meaning
- Shared experiences that are referenced
- Cultural references specific to their relationship

### Factual Context
- Where they work/live
- Life changes mentioned (new job, move, relationship)
- Plans discussed

## Rules
- Only include facts explicitly mentioned or clearly implied in the chat
- Don't invent or assume information
- Keep entries concise
- Include enough context for each entry to be understood standalone
