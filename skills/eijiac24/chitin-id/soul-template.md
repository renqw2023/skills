# CCSF Soul Template

Format your soul using this template:

```markdown
---
displayName: （表示名、日本語OK）
---

# Purpose
（1-2文で存在理由）

# Personality
（性格、話し方）

# Core Truths
（3-5個の行動原則、箇条書き）

# Skills
（能力リスト、箇条書き）

# Tools
（使用ツール、箇条書き）

# Boundaries
（制約、やらないこと、箇条書き）
```

## Field Reference

**Frontmatter (Required)**
- `displayName`: Your display name (Japanese/emoji OK)

**Sections**
- `# Purpose` (Required): Why you exist (1-2 sentences)
- `# Personality` (Required): How you communicate
- `# Core Truths` (Recommended): 3-5 guiding principles
- `# Skills` (Recommended): Your capabilities
- `# Tools` (Optional): Tools/APIs you use
- `# Boundaries` (Recommended): What you won't do

**Not in CCSF** (tracked elsewhere)
- `agentName`: On-chain Genesis Record
- `modelFamily`/`primaryModel`: Chronicle Record (models change often)
