# Track Performance

Track Uniswap LP position performance: identify which positions need attention, are out of range, or have uncollected fees.

→ **[SKILL.md](SKILL.md)** — Full skill specification and workflow.

## Installation

Install into Claude Code or Cursor with:

```bash
npx skills add https://github.com/wpank/Agentic-Uniswap/tree/main/.ai/skills/track-performance
```

Or via Clawhub:

```bash
npx clawhub@latest install track-performance
```

## When to use

Use this skill when:

- You have **existing LP positions** and want a quick health check.
- You need to know which positions are **out of range or underperforming**.
- You want to see where **uncollected fees** have built up.

## Example prompts

- "Track the performance of all my Uniswap LP positions and highlight those out of range."
- "Show which positions have the largest uncollected fees and how long they've been uncollected."
- "Give me a performance dashboard for my Uniswap LP activity over the last month."
