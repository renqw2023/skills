# Research Token

Research a token's Uniswap liquidity, volume profile, pool distribution, and risk factors.

→ **[SKILL.md](SKILL.md)** — Full skill specification and workflow.

## Installation

Install into Claude Code or Cursor with:

```bash
npx skills add https://github.com/wpank/Agentic-Uniswap/tree/main/.ai/skills/research-token
```

Or via Clawhub:

```bash
npx clawhub@latest install research-token
```

## When to use

Use this skill when:

- You want **due diligence on a token's tradability** on Uniswap.
- You need to know **where liquidity lives** (which chains and pools) and how deep it is.
- You care about **volume, volatility, and basic risk flags** before trading or LPing.

## Example prompts

- "Research PEPE on Uniswap—where is its liquidity and how much volume does it see?"
- "Before I trade, show me the Uniswap liquidity and risk profile for this token address on Base."
- "Compare the Uniswap presence of UNI across mainnet, Arbitrum, and Optimism."
