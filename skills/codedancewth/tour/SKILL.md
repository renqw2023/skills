# shenzhen-tour — 深圳本地游玩助手

## 🎯 能力
当用户询问深圳游玩相关问题时，自动：
- 获取实时深圳天气（通过 `weather` 技能）
- 推荐适合当前天气的景点/商场（8个精选地点）
- 生成结构化一日游计划（含时间线与小贴士）

## 🔍 触发关键词
- 深圳游玩 / 深圳旅游 / 今天去深圳玩  
- 深圳有什么好玩的 / 深圳推荐 / 深圳打卡  
- 去深圳 / 深圳一日游 / 深圳周末去哪儿

## 📦 依赖
- `weather`：获取天气数据（内置，无需额外配置）

## 💡 使用示例
> 用户输入：*“深圳今天适合去哪玩？”*  
> 输出：天气 + 3–4 个推荐地点 + 时间线计划
